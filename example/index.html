Mirotone

Mirotone is a small & leightweight skeleton UI Kit that tries to match Miro styles for building apps and plugins. The package contains a CSS file which can be used for any project.

How does it work

Mirotone uses the BEM naming conventation. You define the type of CSS and add a modifier to it.

Example: <button class="button button-primary"> Primary button </button>

Icons

You can use the Miro icons by adding class .icon + .icon-eye, eye in this case is the icon name. You can find all icons in the dist/icons folder. 

<div class="icon icon-eye"></div>

Buttons

When using buttons please add for accessibility. You can find two sizes of buttons: .button and .button-small

Primary button

Use the .button class to trigger the default button.

Use the .button-primary to add modificator for color.

<button class="button button-primary">Primary button</button>
 <button class="button button-primary button-small">Primary small</button>

Secondary button

Use the .button class to trigger the default button.

Use the .button-secondary to add modificator for color.

<button class="button button-secondary">Secondary button</button>
 <button class="button button-secondary button-small">Secondary small</button>

Danger button

Use the .button class to trigger the default button.

Use the .button-danger to add modificator for color.

<button class="button button-danger">Danger button</button>
 <button class="button button-danger button-small">Danger small</button>

Secondary border button

Use the .button class to trigger the default button.

Use the .button-secondary-border to add modificator for color.

<button class="button button-secondary-border">Secondary Border button</button>
 <button class="button button-secondary-border button-small">Secondary Border small</button>

Danger border button

Use the .button class to trigger the default button.

Use the .button-danger-border to add modificator for color.

<button class="button button-danger-border">Danger Border button</button>

Disabled button

Use the disabled="" attribute for disabling buttons.

<button class="button button-primary" disabled>Primary button</button>
 <button class="button button-primary button-small" disabled>Primary small</button>

Icon button

Use the .button-icon class + .icon-{name} to render an icon button.

Add the .button-icon-small if you want to add a smalller icon button.

<button class="button-icon icon-eye"></button>

Textarea

Use a .form-groupto group the textarea with label and status if needed.

<div class="form-group">
  <label>Textarea</label>
  <textarea
    class="textarea language-html"
    placeholder="Placeholder text"
    spellcheck="true"
    rows="4"
  ></textarea>
 </div>

Input

Use a .form-groupto group the input with label and status if needed.

You can find two sizes of inputs: .input and .input-small

For .input-small combine it with a .form-group-small as wrapping style. 
<div class="form-group">
  <label>Input</label>
  <input class="input" type="text" placeholder="Enter something"/>
 </div>
                      
<div class="form-group-small">
  <label>Input</label>
  <input class="input input-small" type="text" placeholder="Enter something"/>
